Le Burger Lillois.

Présentation

Le Burger Lillois est un site qui propose de découvrir les meilleures adresses de burgers de Lille. L'utilisateur peut sélectionner un quartier (Vieux-Lille, Lille-Centre, Lille-Sud) et découvrir les différents restaurants. Afin d'affiner son choix, il peut également filtrer sur les notes. Ce projet a été réalisé dans le but de travailler la construction d'une API, la définition des routes et les appels pour retourner les résultats.
